## 🚀 使用方法

### 启动服务器
```bash
# 方法1：使用启动脚本
./start_server.sh

# 方法2：直接运行Python服务器
python3 server.py

# 方法3：简单HTTP服务器
python3 -m http.server 8080
```

### 访问地址
- **本地访问**: http://localhost:8080
- **局域网访问**: http://172.16.4.47:8080（您的实际IP）

### 操作步骤
1. 在浏览器中打开上述地址
2. 基本使用：
   - 点击底部的"🎲 随机推荐"按钮获取推荐结果
   - 点击"⚙️ 筛选选项"进行个性化设置
3. 筛选选项设置：
   - 设置城市数量（1-100个）
   - 选择特定省份进行筛选
   - 控制是否允许同省多个城市
   - 开启/关闭天气信息显示
4. 查看推荐结果和统计信息
5. 选择困难时可使用随机抽奖功能
