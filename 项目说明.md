## 🚀 使用方法

### 启动服务器
```bash
# 方法1：使用启动脚本
./start_server.sh

# 方法2：直接运行Python服务器
python3 server.py

# 方法3：简单HTTP服务器
python3 -m http.server 8080
```

### 访问地址
- **本地访问**: http://localhost:8080
- **局域网访问**: http://172.16.4.47:8080（您的实际IP）

### 操作步骤
1. 在浏览器中打开上述地址
2. 输入想要的城市数量（1-100）
3. 点击"随机推荐"按钮
4. 查看推荐结果和统计信息
